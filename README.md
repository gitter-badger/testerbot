# testerbot
Automatic Front-End Testing

# Setup

```
npm i testerbot -D
```

# Usage

```
testerbot [--reporters jest-dashboard]
```

# Features

## Automatic Crawling

## Pre-defined Test Cases

## Multiple pages (multiple urls) running all tests

## Disable tests (Warning results)

## Tags

## Authentication support

## Scenarios
